### NYFS Dashboard

**Public food safety intelligence product for NYC restaurant inspection data**

Built a data product with a scheduled refresh pipeline, API layer, and interactive frontend so inspection data can be explored without hitting source APIs on every request.

**Highlights**
- Daily data refresh with GitHub Actions
- FastAPI backend for clean data access
- React and MapLibre frontend for map-first exploration
- Processed public NYC Open Data inspection records

**Tech Stack**: `Python` `FastAPI` `React` `MapLibre` `GitHub Actions`

**[View Project](https://github.com/ShahbazShaikh96/nyfs-dashboard)**

</td>

### Loan Default and Loss Prediction

**Two-stage machine learning framework for credit risk analytics**

Modeled default probability and loss severity using classification and regression workflows across multiple statistical and machine learning approaches.

**Highlights**
- Default classification and loss severity prediction
- Logistic Regression, Elastic Net, Ridge, PLS, Random Forest, XGBoost
- Reproducible notebook, report, and presentation assets
- Built for applied predictive analytics decision support

**Tech Stack**: `Python` `R` `Jupyter` `XGBoost` `Random Forest`

**[View Project](https://github.com/ShahbazShaikh96/loan-default-loss-prediction)**

</td>
</tr>

<td width="50%" valign="top">

### Hybrid Indexing for Big Data Query Routing

**Metadata-guided indexing prototype for distributed query efficiency**

Compared centralized, fully distributed, and hybrid indexing strategies for selective lookup across partitioned big-data-style records.

**Highlights**
- Benchmarked 100,000 records across 8 partitions
- Tested point, range, and keyword queries
- Used coordinator metadata to prune irrelevant partitions
- Designed around latency, message volume, and contacted-node metrics

**Tech Stack**: `Python` `Benchmarking` `Distributed Systems` `Data Structures`

**[View Project](https://github.com/ShahbazShaikh96/hybrid-indexing-big-data)**

</td>
<td width="50%" valign="top">

**Highlights**

- Useful bridge between ad operations and applied analytics

**Tech Stack**: `JavaScript` `Ad Tech` `Contextual Targeting` `Product Analytics`
